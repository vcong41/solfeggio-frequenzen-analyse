## Was die Wissenschaft wirklich sagt

Hier wird es ehrlich. Für die meisten Frequenzen gibt es keine robusten klinischen Studien. Die Ausnahme ist 528 Hz, die tatsächlich untersucht wurde.

Eine 2018 im *Journal of Evidence-Based Integrative Medicine* veröffentlichte japanische Studie mit neun Erwachsenen fand, dass Musik mit 528 Hz den Speichel-Cortisolspiegel (Stresshormon) senkte und die autonome Nerventätigkeit beruhigte – im Vergleich zu 440 Hz. Eine weitere Arbeit von Daylari et al. (2019) deutet auf einen Einfluss auf Hormonsysteme hin. Das sind kleine Studien, keine Beweise für „DNA-Reparatur". Wer das behauptet, überinterpretiert.

Die Aussage „528 Hz repariert DNA" ist eine esoterische Lesart, die aus einer bestimmten Interpretation der Arbeit von Horowitz stammt, nicht aus mainstream-Wissenschaft. Realistisch bleibt: Bestimmte Frequenzen können Entspannung und Stimmung verbessern – ähnlich wie andere Entspannungsmusik auch, aber mit dem Unterschied, dass ein reiner Ton als konstanter Reiz wirkt.

**Einfach gesagt:** Wissenschaftlich am besten belegt ist 528 Hz für Stressreduktion. Für alles darüber Hinausgehende gibt es Erfahrungsberichte, aber keine harten Beweise.
